# Stage 2 — Agent 语义匹配 Prompt 模板

> 本文件是 harness 侧派发 Agent（`subagent_type: general-purpose`）时使用的
> prompt 模板。Agent 语义匹配**不能由 Python 脚本调用**（Agent 是 harness
> 工具），因此 Slice 4 只产出此模板 + 编排纯函数（`scripts/stage2_agent.py`
> 的 `build_batches` / `write_prompts`）。运行步骤见 `RUN.md`。

## 任务

对每一个大绿本（2026 招生计划）本科专业行，从其同校候选集（近三年录取
线差统计的同校历史专业，已按基础专业名/核心名预筛）中，找出**唯一的**那
一个语义对应历史专业；若确无对应，回答 `null`。

## 输入（每条 item）

```json
{
  "school": "<基础校名>",
  "school_cat": "<招生类别，普通为空串>",
  "major": "<大绿本归一化专业全名>",
  "core": "<核心名，去全部括号>",
  "subject": "<选科要求>",
  "src_row_idx": <int>,
  "candidates": [
    {"major": "<近三年专业全名>", "J": <float|null>, "T": <float|null>, "core": "<核心名>"},
    ...
  ]
}
```

## 输出（每条 item 一行 JSON，写入 `batch_NN_result.jsonl`）

```json
{
  "src_row_idx": <与输入相同>,
  "school": "<与输入相同>",
  "major": "<与输入的大绿本 major 相同>",
  "match": "<候选 major 字符串，须逐字等于某 candidate.major；或 null>",
  "J": <float|null>,
  "T": <float|null>,
  "reason": "<≤30 字的六要素对齐理由；null 时写无对应原因>"
}
```

## 判定规则（必须遵守）

1. **六要素差异化**（spec §5.2）——下列要素必须对齐才算「同一专业」：
   - 核心名（去全部括号的基础专业名）
   - 性别（男/女；仅当大绿本与候选都有性别括号时才视为差异化）
   - 招生类别（普通/中外合作/地方专项/走读/边防预科…）—— 大绿本 item 的
     `school_cat` 与候选所属的同校轨道必须一致
   - 校区（仅当大绿本全名或候选全名含明确校区括号时差异化）
   - 方向/培养模式（拔尖/严济慈/师范/创新计划…）
   - 合作轨道（中外合作办学/校企合作）
2. **选科 = 非差异化**：选科政策 2023→2025 跨年漂移，选科差异**不**阻断
   匹配。若选科不同仍匹配，无需在 reason 中提及（日志会由编排层补
   「选科政策漂移，已忽略」）。
3. **忽略类客观要求括号**：色盲/色弱/视力/身高/体重/体检标准/语种/单科成
   绩/年龄等（见 `scripts/constants.py` 的 `IGNORE_BRACKET_KEYWORDS`）**完全
   忽略**，不参与差异化判断。
4. **至多 1 对应**：每个大绿本专业至多匹配一个候选；无法唯一确定时回答
   `null` 并在 `reason` 写明歧义（如「同校两个方向括号都兼容」）。
5. **大绿本名远比近三年详细**：近三年通常只有核心名或核心名+一个方向括号；
   大绿本常含子专业清单、校区、国际交流长文。因此「候选的差异化括号是大
   绿本全名的子集」即视为对齐，不要因大绿本多出的描述性括号而拒绝匹配。

## reason 写法（≤30 字）

- 命中：点出对齐的核心要素，如「核心名同、拔尖方向对齐」「合作轨道一致、
  核心名同」。
- 无对应：写明原因，如「无对应历史专业」「同校两候选方向括号都兼容，歧义」
  「招生类别不同轨道无候选」。

## 严格禁止

- `match` 字段不得返回候选集 `candidates` 之外的字符串（编排层会拒绝并抛
  `Stage2ContractError`）。
- 不得编造 `J` / `T`：必须从所选候选的 `J` / `T` 原样回填；`match=null` 时
  `J`/`T` 必须为 `null`。
- 同一 `src_row_idx` 不得在结果中出现两次（编排层拒绝重复）。
- `reason` 不得为空字符串（编排层拒绝）。
