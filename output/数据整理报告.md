# 山东省 2026 高考录取数据整理报告

> 生成日期：2026-06-24｜数据：大绿本（2026 招生计划）× 近三年录取线差（2023-2025）
> 本文档说明 `output/` 下各表格的作用、关系与口径，便于查阅与人工核验。

## 一、整理了什么

把 **2026 大绿本的每个本科专业**（主键）与 **近三年（2023-2025）录取线差**（外键）按专业名一一对应，在每个专业行尾补上「近三年统计线差 J」与「线差标准差 T」+ 说明日志。因每年专业增删/改名/选科政策漂移，无法直接字符串对应——经「严格→粗筛→语义→复核」多级匹配，存疑/无法匹配的单独成表。

**最终主表**：25474 个本科专业，其中 **19061 确定匹配**（带 J/T）、3450 新增估算（带 J/T）、其余单列待人工。

## 三、表格之间的关系

```
                  大绿本本科专业（25474，主键）
                          │
        ┌─────────────────┼─────────────────┐
   确定匹配(19061)     新增(3746)        特殊/存疑/改名(2942)
   严格+粗筛+语义         │                  │
   （经复核确定）    新增专业.xlsx      特殊情况.xlsx
        │           （估算 J/T）      （复核存疑+未归类）
        │                │                  │
   主表 J/T 填值     改名校专业(225)→ 学校改名表.xlsx（13校）
                          │
            被删旧专业.xlsx（近三年侧：16827，学校仍在但2026缺）
```

- **主表**（分层版/扁平版）：每个本科专业一行，确定匹配的填 J/T，其余（新增/特殊/改名）J/T 留空但日志说明去向。
- **边界表**（新增/被删/特殊/改名/新增校/停招）：把"非确定匹配"的部分按原因分类单列，供人工核验。被删/新增校/停招是**学校或专业层面**的差集；特殊/改名是**专业层面**的存疑。

## 五、关键口径

- **线差** = 录取低分 − 当年一段线（2023=443 / 2024=444 / 2025=441）。统计线差 = 可用年份均值；标准差 = 同年份总体标准差。
- **精度**：所有 J/T 舍入到 **2 位小数**（匹配近三年源表精度，如 T=12.73）。matched 行保留源值；新算的（提前批/估算）舍入 2 位。
- **T（标准差）**：单年历史 → 无标准差（留空 + 日志说明）；新增估算 → J 与 T 都按同校同选科均值估；退化2（无历史）→ J/T 都空。
- **主表零错配（precision-first）**：仅"严格精确匹配"构造确定直接入主表；所有判断型（粗筛/语义）经**二次 agent 复核**，只有"确定"才留主表，"存疑"移特殊表。保证主表无错配。
- **选科非差异化**：2023→2025 选科政策漂移（物理→物理和化学），匹配时不因选科差异判不同专业（日志标注漂移）。
- **专科全排除**：扁平版剔除；分层版标注「不在本次整理范围」。
- **范围**：仅本科、仅提前批 + 常规批一段。

## 六、列说明

**主表行尾追加列**（分层版/扁平版，不覆盖大绿本原 12 列）：
- `近三年统计线差`（J）、`近三年线差标准差`（T）、`匹配日志`（阶段+原因，详见第四节）。

**新增专业.xlsx 列**：学校/专业/选科/统计线差估算/线差标准差估算/退化级别(0/1/2)/样本量/日志。
**被删旧专业.xlsx 列**：学校/招生类别/专业/近三年统计线差/近三年线差标准差/日志。
**学校改名表.xlsx 列**：2026新校名/候选旧校名/置信度/2026本科专业数/备注(网查)/人工已核验。
**特殊情况.xlsx 列**：src_row_idx/学校/招生类别/专业/核心名/选科/批次/日志。
