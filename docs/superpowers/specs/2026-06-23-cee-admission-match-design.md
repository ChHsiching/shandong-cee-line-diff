# 山东省高考录取数据匹配整理 — 设计文档（Spec）

- 日期：2026-06-23
- 状态：**v3（grilling 后修订 + 学校改名分支；待 prototype 验证阈值后定稿）**
- 范围：**纯数据管线**，无 UI / 后端（baoyu-design、visual-companion 不适用）
- 目的：把 2026 大绿本招生计划（主键）与近三年录取线差（外键）按专业名语义一一对应，输出带线差与日志的整理表，并沉淀为可年度复用的 agent skill。
- 变更：v2 整合 grilling 六项决策；**v3 新增「招生类别」差异化项 + 校名类别剥离归一化 + 学校改名分支（单独列表+人工+网查备注）**。

## 1. 背景与目标

2026 高考已结束、6/25 出分。为辅助志愿填报，需把今年的招生计划（大绿本）与过去三年各专业录取线差关联起来。因每年招生计划会增删/改名专业、甚至**学校改名/转设**，无法直接按字符串对应——需**语义匹配**。

核心目标：以大绿本专业名为**主键**，在合并后的近三年数据中找**唯一**对应专业，在大绿本每个专业行尾追加「近三年统计线差(J)」「近三年线差标准差(T)」「匹配日志」三列。保证：近三年数据中不存在语义歧义，每个大绿本专业至多对应 1 个。

## 2. 数据资产（实证结构，2026-06-23）

### 2.1 大绿本 `山东省2026年大绿本招生计划.xlsx`（主键源）
- 单 sheet，27610 数据行，12 列。**分层结构**：批次头行 / 小标题行 / 学校行 / 专业行。
- 列：A 批次 · B 小标题（招生类别）· C 学校代码 · D 学校名 · E 代号 · **F 名称（专业名，匹配键）** · G 选考科目要求 · H 学制 · I 计划数 · J 学校备注 · K 年收费 · L 整行校准。
- **专业行判别**：E(代号) 与 F(名称) 同时非空。
- 批次(A)：`4.常规批`(25591) · `1.提前批A类`(1249) · `2.提前批B类`(765) · `3.提前批—飞行技术(军队)`(5)。
- 专业行总数 25655；按批次：常规批 23887 / 提前批A类 1139 / 提前批B类 627 / 飞行技术 2。
- 小标题(B) 17 类（=招生类别）：普通计划(23114) · 中外合作办学(1551) · 高校专项计划(613) · 军事类(487) · 公安政法类(409) · 地方专项计划(293) · 综合评价招生(247) · 定向培养军士生(专科)(229，含非专业行；专业行 181 见 §3) · 省属公费师范生(164) · 航海类(64) · 边防军人子女预科班(19) · 飞行技术(民航)(15) 等。
- 选考科目要求(G) 为干净单值：物理和化学(15151) · 不限(7138) · 物理(1736) …
- 校名 1206 所（剥类别后缀）。

### 2.2 近三年 `近三年学校批次专业线差统计.xlsx`（常规批一段 J/T 源 + 部分 提前批）
- sheet `统计结果`：46648 行，29 列。关键列：A batch · B schoolcode · C schoolname · D majorname · E subject · F remarks · **G 基础专业名** · H 是否括号专业 · **I 括号内容** · **J 统计线差** · … · **T 线差标准差** · … · Y 同基础专业版本数。
- batch 取值：`常规批一段线`(28269) · `常规批二段线`(17554，**删除**) · `提前批`(825，**弃用**)。
- **subject(E) 37.5% 为多值**（如「物理 | 物理和化学」），系 2023→2025 选科**政策漂移**所致。
- **校名常把「招生类别」拼进校名**（如「三亚学院(中外合作办学)」「山东中医药大学(地方专项计划)」「山东建筑大学(走读)」「桂林电子科技大学(边防军人子女预科班)」）——须剥除后才能与大绿本校名对齐。
- 辅助 sheet：`括号语义判断`、`同基础专业多版本`、`较2025浮动提醒`、`说明`。
- `说明` 口径：线差=`low_score − 山东一段线`，2023=443 / 2024=444 / 2025=441；预测=可用年份线差均值（单年直接用该年）。

### 2.3 提前批补充表 `山东省高考提前批录取数据.xlsx`（提前批 J/T 计算源）
- 单 sheet，1900 行，43 列。列：A 批次名称 · B 招生类别 · C 院校代码 · D 院校名称 · E 专业代码 · F 专业名称 · G 选科 · H-S 三年录取{人数/高分/均分/低分} · …
- **无现成线差**，仅有原始分。批次名称：`本科提前批A类`(1161) · `本科提前批B类`(546) · `专科提前批`(193，**删除**)。
- 本科 A+B 共 1707 行；每行拥有「录取低分」的年份数分布：3 年全有 866 / 2 年 254 / 1 年 585 / 全无 2。

## 3. 锁定的关键决策

| 项 | 决策 | 依据 |
|----|------|------|
| 范围 | 仅 提前批 + 常规批一段，**仅本科** | 用户 |
| 提前批 AB 类 | **无差别**，合并为同一组 | 用户（AB 仅为填报观察便利） |
| 提前批数据来源 | **以补充表为准**；近三年 825 条 `提前批` 弃用（合并前先重叠验证） | 用户 |
| 提前批线差 | 自算 `录取低分 − 当年一段线`（443/444/441）；统计线差=可用年份均值；标准差同口径 | 用户（grilling 确认低分） |
| 专科排除 | 全排除；大绿本=小标题含「专科」(181)；近三年=删二段后一段线已纯本科 | 用户 + 实证 |
| **管线结构** | **三段式**：Stage1 严格(脚本) → Stage1.5 粗筛自动接受(脚本) → Stage2 语义(agent,禁脚本) → Stage3 边界 | grilling Q1（严格仅 ~60%） |
| **粗筛自动接受判据** | 差异化签名与同校**唯一**候选完全对齐才自动接受；否则交 agent | grilling Q2（保守） |
| **选科** | **非差异化**（政策漂移，记日志）；新增估算用选科集合包含关系 | grilling Q3（37.5% 多值） |
| **招生类别**（v3 新增） | **差异化项**（第 6 要素）：普通/中外合作/地方专项/高校专项/走读/边防预科/航海/公费师范… 不同轨道不同分数线；从校名剥除作标记 | 实证：近三年拼校名、大绿本在小标题(B) |
| **新增估算退化** | 同校同选科均值 → 同校全专业均值(标注) → 整校无历史入特殊表不估算 | grilling Q5 |
| **被删专业范围** | 仅「2026 仍招生的学校」里近三年有、2026 缺的专业；整校缺席另列 | grilling Q6 |
| **学校改名**（v3 新增） | **单独列表 + 人工审核**；不自动跨改名重匹配；全部其他数据处理完后，**最后逐个上网查询写入备注** | grilling Q7 |
| **输出形态** | 分层版 + 扁平专业表，两者同源 | grilling Q4 |
| 源文件 | **只读，永不修改**；移到 `data/`（git mv） | 用户 |

## 4. 合并后的数据模型

### 4.1 统一历史表（Stage 0 产出，匹配右侧/外键池）
- 近三年 `常规批一段线`：28269 行（J/T 已算好）。
- 补充表 `本科提前批A+B`：1707 行（AB 合并，batch 记 `提前批`，**现场计算 J/T**）。
- 合计 ≈ **29976 行**。Schema（对齐近三年）：`batch, schoolcode, schoolname(剥类别后), 基础校名, 招生类别, majorname, 基础专业名, 括号内容, subject, 统计线差, 线差标准差, 可用年份数, 来源表`。
- 提前批 J/T 计算式：`year线差 = 录取低分 − 当年一段线`；`统计线差 = mean(有数据年份线差)`（单年用该年）；`线差标准差 = std(...)`（单年为空）。

### 4.2 大绿本本科专业表（Stage 0 产出，匹配左侧/主键池）
- 大绿本专业行排除小标题含「专科」(181)。常规批 23887 ＋ 提前批(A类 1139 ＋ B类本科 446) ＋ 飞行技术 2 ≈ **25474 行**。保留原批次/小标题(招生类别)用于回填；提前批 A/B 在匹配池合并。

## 5. 匹配键规则（核心）

### 5.1 归一化（精确匹配用）
- `NFKC` 全角↔半角（，→, （→( 全角数字字母）；去全部空白；括号风格统一。
- **校名归一化**：剥除「招生类别」括号后缀（中外合作办学 / 地方专项计划 / 高校专项计划 / 走读 / 边防军人子女预科班 / 定向 / 公费师范 / 航海 …），类别作为差异化标记单独提取（见 §5.2）。实证：不剥则两表校名大量对不上（如「三亚学院」≠「三亚学院(中外合作办学)」）。
- 专业名归一化同上 + 剥「忽略类」括号（见 §5.3）。

### 5.2 差异化签名（决定"是否同一专业"，六要素）
1. **核心名**（去括号基础专业名）
2. **性别**（男/女）——同名按性别拆两条独立匹配（不同分数线）
3. **校外合作**（中外合作办学 / 校企合作 / 合作国别）
4. **校区**（独立校区 / 某校区，如 东北大学 vs 东北大学秦皇岛分校）
5. **方向/描述**（如 量化投资、含电力储能方向）
6. **招生类别**（普通计划 / 中外合作 / 地方专项 / 高校专项 / 走读 / 边防预科 / 航海 / 公费师范…）——来源：大绿本「小标题(B)」、近三年/补充表「校名/招生类别后缀」

### 5.3 忽略类（不影响是否同一专业）
身高 / 体重 / 色盲 / 色弱 / 视力 / 体检标准 / 合格 / 语种 / 单科成绩 / 年龄 / `特殊类型招生控制线` 等考生客观要求——两年要求不同也不算不同专业。性别是唯一例外。

### 5.4 选科 = 非差异化（grilling Q3）
选考科目要求**不作为差异化项**：2023→2025 选科政策漂移使 37.5% 近三年行选科跨年多值，选科不是稳定身份属性。匹配只对齐 §5.2 六要素签名；选科差异记日志「选科政策漂移，已忽略」。真正并发不同选科轨道（罕见）→ 唯一性失败 → 落 agent。新增估算「同选科」用集合包含关系。

### 5.5 辅助字段
优先用近三年 `基础专业名`(G) + `括号内容`(I) 做核心名/区分类识别；`括号语义判断` sheet 可参考。

## 6. 管线设计（三段式 + 改名分支）

### Stage 0 — 预处理合并（脚本，只读源，不改源文件）
- 删 `常规批二段线`；弃 825 `提前批`（先重叠验证：确认 `(schoolcode, 归一化majorname)` 均被补充表覆盖；独有行单列待人工，不静默丢）。
- 补充表算线差并入，AB 合并、删专科提前批。
- **校名剥类别后缀**、提取招生类别标记。
- 大绿本排除小标题含「专科」；导出本科专业表（带招生类别）。
- 产出统一历史表 + 大绿本本科专业表（落 `intermediate/`）。

### Stage 1 — 严格匹配（脚本）
- 键 = `(归一化基础校名, 招生类别, 剥忽略类括号后的归一化全名)`，在统一历史表精确查找。
- 命中取 J/T，日志 `严格匹配：归一化专业名+招生类别一致`。实测覆盖 ~60%（剥类别后预计上升，prototype 复测）。

### Stage 1.5 — 粗筛自动接受（脚本，grilling Q1+Q2）
- 对 Stage 1 未命中者：抽取**六要素差异化签名**，与同校候选签名比对。
- **当且仅当签名与唯一一个候选完全对齐** → 自动接受，日志 `粗筛匹配：签名唯一对齐（<简述>）`。
- 方向措辞模糊 / 签名多义 / 缺候选 → 交 Stage 2。目标覆盖率 ~75-80%。阈值 → **prototype 校准**。

### Stage 2 — 语义匹配（**agent，禁脚本**，grilling Q1）
- 对 Stage 1.5 未命中者，**并行 agent** 逐个推理：输入大绿本专业（校名+全名+选科+招生类别）+ 同校候选集（按基础专业名预筛）。
- agent 输出：唯一对应历史行（或「无对应」）+ J/T + 简短日志。每个至多 1 个。批/并发/prompt → 实现计划阶段。

### Stage 3 — 边界处理
- **新增专业**（大绿本有、历史无；grilling Q5 逐级退化）：
  - 退化0：同校 + 选科集合包含 的历史专业 `统计线差` 均值；日志 `新增专业：估算=同校同选科(<n>)均值=<值>`。
  - 退化1：同校无同选科 → 同校全专业均值；日志 `…退化=同校全专业均值(无同选科)(<n>)=<值>`。
  - 退化2：整校无历史 → 入特殊表，J 留空；日志 `新校/无历史，无法估算`。
  - 行加特殊标记 + 单列成表。
- **被删旧专业**（grilling Q6）：仅「2026 大绿本仍存在的学校」里近三年有、2026 缺的专业；单列成表，日志 `近三年有、2026 大绿本无`。
- **学校改名/转设**（grilling Q7，v3 新增）：
  - 检测改名候选：Stage 0 后「大绿本独有校」(59) 与「历史独有校」(58) 取交集语义配对（**字符串相似度不可靠，须 agent 语义判断**）。
  - 写「学校改名表」：2026新校名 / 候选旧校名 / 置信度 / 该校2026本科专业数 / 备注。**不自动跨改名重匹配**——这些学校的专业在主产出里 J/T 留空、日志 `疑似改名校(见改名表)，待人工核验`。
  - **最后一步（全部其他数据处理完成后）**：逐个上网查询这些学校（WebSearch），把实际改名/转设情况写入备注（旧名、更名时间、是否同源等），供人工据此手动关联。
  - 未配对的大绿本独有校 → 「新增校表」；未配对的历史独有校 → 「停招/消失校表」（含独立学院转设消失）。
- **特殊/无法处理**：单列成表，日志写明原因。
- **飞行技术(军队) 2 行**：归入提前批池参与匹配；不成则入特殊表。

## 7. 输出产物（`output/`）
- `大绿本_附线差_分层版.xlsx`：**完整复制大绿本**（保留批次头/小标题/学校行分层）+ 行尾追加 `近三年统计线差` / `近三年线差标准差` / `匹配日志`（仅专业行填值）。**不覆盖**原列（含原 J 学校备注）。
- `大绿本_附线差_扁平版.xlsx`：仅专业行，每行一专业 + 全字段 + 招生类别 + J/T + 日志，便于排序/筛选/下游分析。
- `新增专业.xlsx`（估算+退化标注）/ `被删旧专业.xlsx`（仅学校仍在）/ `学校改名表.xlsx`（人工+网查备注）/ `新增校表.xlsx` / `停招消失校表.xlsx` / `特殊情况.xlsx`（各带日志列）。
- `README.md` 文档化全过程。

## 9. 日志列规约（简短备注，便于人工核验）
- 严格：`严格匹配：归一化专业名+招生类别一致`
- 粗筛：`粗筛匹配：签名唯一对齐（<简述>）`
- 语义：`语义匹配：<六要素对齐理由>`
- 选科漂移：附加 `选科政策漂移，已忽略`
- 新增：`新增专业：估算=同校同选科(<n>)均值=<值>` / `…退化=同校全专业均值…` / `新校/无历史，无法估算`
- 被删：`近三年有、2026 大绿本无`
- 改名：`疑似改名校(见改名表)，待人工核验`
- 停招：`学校未在 2026 招生`
- 特殊：`无法匹配：<原因>`
- 弃用验证：`近三年提前批，已被补充表覆盖` / `独有，待人工核验`

## 11. 待 prototype 验证的开放点（grilling 后剩余）
- 校名类别剥离 + 六要素归一化后的**精确匹配命中率**复测（基线 ~60%，预期上升）。
- 六要素差异化签名抽取（性别/合作/校区/方向/招生类别）的规则准确率与漏抽/误抽。
- Stage 1.5「签名唯一对齐」的方向措辞容忍度阈值 → 校准到 ~75-80% 覆盖且低误配。
- 改名校语义配对的 agent 准确率（字符串相似度已证不可靠）。
- agent 语义匹配的批大小/并发/prompt 模板（实现计划阶段）。
